# 🚀 DataPilot

AI-assisted data auditing and cleaning copilot built with Python and Streamlit.

DataPilot helps analysts quickly inspect, understand and clean datasets through automated quality auditing and smart insights.

## ✨ Current Features

- CSV Upload
- Automated Data Auditing
- Missing Value Detection
- Duplicate Detection
- Data Quality Score
- Cardinality Analysis
- Smart Insights Generation

- ## 🛡️ Privacy-First Architecture

DataPilot was designed with enterprise and LGPD-aware workflows in mind.

- Local data auditing using pandas/statistical rules
- Optional AI interpretation layer
- No raw sensitive data required for insights generation
- Future support for local LLMs (Ollama/Llama)

The platform follows a hybrid architecture where AI acts as an interpretation layer rather than the core execution engine.

## 🛠️ Tech Stack

- Python
- Pandas
- Streamlit
- NumPy
